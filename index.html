What is TypeMorph?

TypeMorph is a simple javascript typing effect engine built for precision and control. It supports html, markdown, chunked typing, backspacing modes, looping, custom cursors, and clean async control. Perfect for documentation sites, portfolios, LLMs, or any dynamic text UI.

Built for developers. Designed for the modern web.
