Backspace

Backspaces the provided number of characters (or all if count is null) from the target element once.

Hello everyone!
const typer = new TypeMorph({
  parent: "target",
  speed: 60,
  backspaceSpeed: 90,
  clearBeforeTyping: false,
});

await typer.backspace(9);
await typer.type("world");
